# akari
akari is a work in progress python program to manage anime artwork. currently it uses [iqdb](https://iqdb.org) to reverse-search images and fetch appropriate tags.

## Requirements
- requests
- BeautifulSoup 4

## Usage
    akari --scan /path/to/directory
    akari --gui
    akari --help
